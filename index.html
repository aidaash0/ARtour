Quick AR (model-viewer)

Having trouble with the AR button?
- Make sure you are using a supported mobile device (Android ARCore, iOS ARKit/Quick Look).
- The page must be served over HTTPS (not HTTP or localhost).
- The model file path must be correct and accessible.
- Try using Chrome (Android) or Safari (iOS).

Immersive Portal (A-Frame)

Use mouse or touch to look around. On supported devices, enter AR to walk through the portal!
